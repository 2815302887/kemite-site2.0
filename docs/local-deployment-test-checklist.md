# 本地部署前自测清单

> 上线前请先在本机完整跑一遍。重点关注三类问题：浏览器控制台报错、接口 4xx/5xx、刷新或重启后数据丢失。

## 启动前准备

1. 执行构建：

   ```bash
   npm run build
   ```

2. 启动本地服务：

   ```bash
   npm run serve
   ```

3. 打开前台：

   ```text
   http://127.0.0.1:3001
   ```

4. 打开后台：

   ```text
   http://127.0.0.1:3001/admin.html
   ```

## 1. 前台首页

- [ ] 首页能正常打开，无白屏。
- [ ] 导航、按钮、页面滚动正常。
- [ ] 浏览器控制台无红色报错。
- [ ] Network 中静态资源无明显 404。

## 2. 3D 模型

- [ ] 首页 3D 区域能正常显示模型。
- [ ] 加载中提示能正常消失。
- [ ] 切换“上一个/下一个”正常。
- [ ] 鼠标交互或旋转效果正常。
- [ ] Network 中 `dist/model-data.js`、`dist/r3f/app.js` 加载成功。

## 3. 产品中心

- [ ] 打开 `products.html`。
- [ ] 产品卡片正常显示。
- [ ] 产品图片正常加载。
- [ ] 产品名称、简介、标签、按钮显示正常。
- [ ] 无产品加载失败提示。

## 4. 产品搜索筛选

- [ ] 输入产品型号或关键词能筛选结果。
- [ ] 切换分类、应用、合金筛选正常。
- [ ] 多条件组合筛选正常。
- [ ] 清空搜索后列表恢复。
- [ ] 无匹配时空状态显示正常。

## 5. 产品详情页

- [ ] 从产品中心点击详情弹窗正常。
- [ ] 点击进入独立详情页正常。
- [ ] URL 中带 `id` 时能找到对应产品。
- [ ] 产品图片、简介、详情、参数、PDF 下载按钮正常。
- [ ] 输入不存在的 `id` 时有异常提示，不白屏。

## 6. 留言提交

- [ ] 打开 `contact.html`。
- [ ] 空表单校验正常。
- [ ] 正常填写姓名、电话或邮箱、留言后提交成功。
- [ ] 提交后页面提示正常。
- [ ] 后端接口 `/api/contact` 返回成功。
- [ ] 重复快速提交时频率限制正常。

## 7. 后台登录

- [ ] 打开 `admin.html`。
- [ ] 未登录时显示登录页。
- [ ] 错误账号密码有失败提示。
- [ ] 正确账号密码能进入后台。
- [ ] 刷新后台页面后登录态仍有效。
- [ ] 点击退出登录后回到登录页。

## 8. 产品新增、编辑、删除、排序

- [ ] 产品列表能分页显示。
- [ ] 搜索后台产品正常。
- [ ] 新增产品后能出现在列表中。
- [ ] 编辑名称、分类、应用、合金、简介、详情、参数后保存成功。
- [ ] 刷新前台产品中心后能看到变化。
- [ ] 上移/下移排序正常。
- [ ] 删除测试产品前有确认弹窗。
- [ ] 删除后前台和后台都不再显示该产品。

## 9. 图片上传

- [ ] 后台产品编辑中上传 PNG/JPG/WebP。
- [ ] 上传成功后图片路径自动回填。
- [ ] 预览图正常显示。
- [ ] 保存产品后前台产品图片正常显示。
- [ ] 上传不支持格式或过大图片时有错误提示。

## 10. PDF 上传

- [ ] 后台产品编辑中上传 PDF。
- [ ] 上传成功后资料路径自动回填。
- [ ] 保存产品后前台详情页出现下载按钮。
- [ ] 点击 PDF 下载或打开正常。
- [ ] 上传非 PDF 文件时有错误提示。

## 11. 图片库 used 禁删

- [ ] 打开后台图片库。
- [ ] 已被产品使用的图片显示 used/使用中状态。
- [ ] 使用中的图片删除按钮不可用或删除被拦截。
- [ ] 未使用图片可以单删。
- [ ] 未使用图片可以批量勾选删除。
- [ ] 删除后刷新图片库，图片不再出现。

## 12. 重启服务后的数据持久化检查

1. [ ] 新增一个测试产品并保存。
2. [ ] 上传一张测试图片和一个 PDF。
3. [ ] 提交一条测试留言。
4. [ ] 停止本地服务。
5. [ ] 重新执行 `npm run serve`。
6. [ ] 再次打开前台产品中心，确认测试产品仍在。
7. [ ] 打开产品详情页，确认图片和 PDF 仍在。
8. [ ] 登录后台，确认测试产品仍在。
9. [ ] 打开留言管理，确认测试留言仍在。
10. [ ] 打开图片库，确认图片 used 状态仍正确。
